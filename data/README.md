# Data Directory

Data files are downloaded and cached here automatically when running the pipeline or notebooks.

## Files Generated

- `prices_2015-01-01_2024-01-01.parquet` — Adjusted OHLCV prices for all tickers (generated by `StockDataCollector.download_prices()`)
- `news_headlines.csv` — Financial news headlines for sentiment analysis (generated by `StockDataCollector.collect_all_news()`)

## Data Collection Methodology

Price data is sourced from **Yahoo Finance** via the `yfinance` Python library, which provides:
- Adjusted close prices (splits and dividends corrected)
- Complete OHLCV (Open, High, Low, Close, Volume)
- 9 years of daily data (2015–2024) for 10 major US equities
- Total: ~22,500 stock-day observations

News data is fetched from **Yahoo Finance's news endpoint** via `yfinance.Ticker.news`. This returns recent headlines with publication timestamps, which are then scored for sentiment by the Claude API.

## Tickers

AAPL (Apple), MSFT (Microsoft), GOOGL (Alphabet), AMZN (Amazon), NVDA (Nvidia), TSLA (Tesla), META (Meta), JPM (JPMorgan), BRK-B (Berkshire Hathaway), UNH (UnitedHealth)

These represent a cross-section of sectors (tech, finance, healthcare) to reduce overfitting to sector-specific patterns.
