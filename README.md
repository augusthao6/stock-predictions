## Multi Model Trading System

A multi-stage machine learning pipeline for stock market direction prediction, combining LSTM temporal modeling, gradient-boosted trees, and Claude AI sentiment analysis in an ensemble framework, evaluated through portfolio backtesting.

## What it Does

This project predicts the daily direction (up/down) of stock returns for 10 major US stocks/equities (AAPL, MSFT, GOOGL, AMZN, NVDA, META, JPM, GS, SPY, QQQ) by fusing three complementary information sources: (1) a custom stacked LSTM neural network that learns temporal patterns from 20-day windows of 21 technical indicators, (2) XGBoost and Random Forest models trained on the same features with a 1-day temporal lag to prevent data leakage, and (3) financial news sentiment scores generated by calling the Claude API on Yahoo Finance headlines. These three prediction streams are combined via a logistic regression meta-learner (stacking ensemble). The final trading signals drive a portfolio simulation that compares the ensemble strategy against buy-and-hold and random baselines using financial performance metrics including Sharpe ratio, maximum drawdown, and annualized return.

## Evaluation

### Classification Results — 10-Ticker Pooled Validation Set (3,300 samples)

| Model | Val Accuracy | Val F1 |
|-------|-------------|--------|
| Random Baseline | 50.0% | — |
| XGBoost | 50.3% | 55.0% |
| Random Forest | 48.7% | 47.6% |
| LSTM | **55.8%** | — |
| **Ensemble (meta-learner)** | **56.7%** | 72.4% |

### Portfolio Backtesting — AAPL Test Period (last 15% of 2017–2026 data, ~312 days)

| Strategy | Annual Return | Sharpe Ratio | Max Drawdown | N Trades |
|----------|--------------|--------------|--------------|----------|
| Buy-and-Hold | −8.95% | −0.385 | −30.43% | 1 |
| Random | −30.78% | −2.001 | −36.59% | 173 |
| Random Forest | −2.50% | −0.361 | −20.07% | — |
| **LSTM** | **+4.98%** | **+0.077** | **−14.35%** | 16 |
| Ensemble | −9.54% | −0.539 | −27.35% | 61 |

LSTM is the only strategy with a positive Sharpe ratio and annual return, outperforming buy-and-hold by ~14 percentage points during a drawdown period by selectively sitting in cash.

### Ablation Study — AAPL Single-Ticker (50-epoch LSTM per condition)

| Condition | Test Accuracy | Notes |
|-----------|-------------|-------|
| No Sentiment | 53.5% | baseline |
| **With Sentiment (Claude API)** | **56.7%** | +3.2 pp — sentiment helps |
| Price Only (5 features) | 52.2% | fewer features hurt |
| Momentum Only (7 features) | 57.1% | best single group |
| Full Features (20 features) | 51.6% | curse of dimensionality |

### Hyperparameter Search — 6 Configurations (30 epochs each)

| Hidden | Layers | Dropout | Val Acc |
|--------|--------|---------|---------|
| 64 | 2 | 0.2 | ~53.5% |
| 64 | 2 | 0.3 | ~53.4% |
| 128 | 2 | 0.2 | ~54.4% |
| **128** | **2** | **0.3** | **~54.8% ← best** |
| 256 | 2 | 0.2 | ~54.3% |
| 256 | 2 | 0.3 | ~54.1% |

### Key Findings

- LSTM achieves positive Sharpe ratio and beats buy-and-hold during the 2024–2026 AAPL test period
- Claude sentiment analysis adds +3.2 percentage points test accuracy vs. no sentiment (ablation)
- Momentum features alone outperform full feature set (20 features) — curse of dimensionality effect
- LSTM max drawdown (−14.35%) is 16 pp lower than buy-and-hold (−30.43%)
- RSI, MACD, and log-return features are top-ranked by XGBoost importance
