# TradeSage: AI-Powered Multi-Model Trading System

A multi-stage machine learning pipeline for stock market direction prediction, combining LSTM temporal modeling, gradient-boosted trees, and Claude AI sentiment analysis in an ensemble framework, evaluated through portfolio backtesting.

## What it Does

TradeSage predicts the daily direction (up/down) of stock returns for major US equities by fusing three complementary information sources: (1) a custom stacked LSTM neural network that learns temporal patterns from 20-day windows of technical indicators, (2) XGBoost and Random Forest models trained on the same features to capture non-linear interactions, and (3) financial news sentiment scores generated by calling the Claude API on news headlines. These three prediction streams are combined via a logistic regression meta-learner (stacking ensemble). The final trading signals drive a portfolio simulation that compares the ensemble strategy against buy-and-hold and random baselines using financial performance metrics including Sharpe ratio, maximum drawdown, and annualized return.

### Key Findings

- Ensemble consistently outperforms every individual model by 1–3% accuracy
- High-volatility market regimes are the primary source of model failures
- RSI, MACD histogram, and Bollinger %B are the top predictive features
- Ensemble strategy achieves better risk-adjusted returns than buy-and-hold in several periods
